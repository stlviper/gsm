Atlassian User Interface library (AUI)

About AUI

AUI is a tailor-made frontend library for creating a UI according to the Atlassian Design Guidelines. It provides JavaScript, CSS and Soy templates for a range of ready-made UI components.

Getting Started

If you're working in an Atlassian product, it probably already has AUI installed as a plugin. If you're working outside an Atlassian product or simply want to do some prototyping, the flat pack is for you.
